Projects

I love to work on side projects in my free time. Here are some things I have worked on:

  • LetsGit is an accessible, cloud-based Git GUI client and code editor in the browser. Lets you easily code and collaborate from anywhere without needing to setup a new development environment.
  • Twitch Plays Tinder is a web app that streams a port of Tinder onto twitch.tv and allows collaborative matchmaking on the platform. Originally inspired by Twitch Plays Pokémon.
  • ScandIn is a Google Glass application to enhance networking events using facial recognition. The Glass application takes a picture, sends it to an Express.js backend, matches the image to a user in the training set, and returns the recognized user's information to a Glass card.
  • 3DSnap allows you to share and view stereoscopic 3D snapchats with a 3DS or mobile phone.
